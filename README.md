# Smart-Retails-management-system
The Smart Retail Management System is a web-app to track product,manage inventory
and generate reports from a central dashboard which was developed using PHP and
MySQL to manage the daily activities of a store efficiently. The system is expected to
help the store owners to manage products, inventory, orders, and sales records digitally. It
replaces manual record-keeping with an automated system, improving accuracy and
saving time.

# System Features
● Admin Login System
● Product Management
● Category Management
● Inventory Management
● Order Management
● Sales Management
● Report Generation
